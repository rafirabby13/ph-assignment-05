Important FAQ

What is the Document Object Model (DOM)?

Document Object Model (DOM) allows developers to interact and manipulate the structure, content and style of web pages.It represents entire HTML as a tree structure.

How do you select an element from the DOM?

There are lots of ways to select an element from the DOM. These are:


  • getElementById()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()
  • getElementsByTagName()
What is event delegation in the context of the DOM, and why is it useful?

If there is multiple children of a parent in HTML, than we use delegation.In delegation, event listener added to the parent rather to children. The event listener can capture events triggered by its children.

How do you manipulate anelement's attributes and styles using the DOM?

Attributes can be manipulated using methods like getAttribute(), setAttribute(), and removeAttribute(). Styles can be directly manipulated using the style property or by managing CSS classes with classList.


Manipulate Using Attributes:

  • element.setAttribute(name, value)
  • element.getAttribute(name)
  • element.removeAttribute(name)

Manipulate Using Styles:

  • element.style.color='red
  • element.style.backgroundColor = "blue"

Manipulate Using Class:

  • element.classList.add(class);
    • element.classList.remove(class);
